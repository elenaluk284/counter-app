# Счетчик (Counter)

Простой счетчик на HTML, CSS и JS.

## Функционал:
- Кнопки плюс/минус меняют значение
- Цвет фона меняется в зависимости от числа:
  - Жёлтый (положительные)
  - Красный (ноль)
  - Зелёный (отрицательные)
- Блокировка кнопок при ±10
- Сообщение при достижении крайних значений

## Запуск:
Откройте файл `index.html` в браузере.
